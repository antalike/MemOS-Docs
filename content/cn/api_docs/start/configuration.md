---
title: 项目配置
---

MemOS 支持以项目为维度管理资源、权限和调用日志。一个项目可以是一款 App、一个 Agent，或者任何需要独立管理资源的模块。

## 1. 新建项目

*   每位新用户默认拥有一个“默认项目”。
    
*   通过控制台新建项目时，输入名称和描述，即可创建属于你的独立项目。
    

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65AKwBZP9Oke/img/6873f613-d9e2-4aae-93d1-49d2116ad526.png)

## 2. 删除项目

*   在拥有多个项目时，可删除不再需要的项目。
    

❕删除项目将清空该项目下的所有记忆、消息及相关数据，该操作**不可恢复**。

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65AKwBZP9Oke/img/b0735f87-ad96-4052-871d-5f2425519ffb.png?x-oss-process=image/crop,x_0,y_0,w_3024,h_1463/ignore-error,1)

## 3. 项目接口密钥

*   每个项目拥有独立的接口密钥列表，用于访问该项目下的所有记忆、消息和数据。
    
*   在控制台左上角切换项目，即可查看对应密钥。
    

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65AKwBZP9Oke/img/c4494ff2-2613-483a-b7a2-7874b193f9c1.png?x-oss-process=image/crop,x_0,y_0,w_3024,h_1462/ignore-error,1)

## 4. 项目调用日志

*   在控制台左上角切换项目，监控接口调用情况与历史记录。
    

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/AJdl65AKwBZP9Oke/img/f964a5db-48c5-4ae6-8358-281167e4410a.png?x-oss-process=image/crop,x_0,y_0,w_3024,h_1467/ignore-error,1)

## 5. 项目知识库

*   在控制台或通过接口创建知识库，添加与本项目关联的知识库。
    

::note
想要了解知识库原理与使用方法，前往<u>知识库介绍页面</u>，按照讲解一步步创建可以与用户记忆一起被召回的知识库文档记忆。
::
